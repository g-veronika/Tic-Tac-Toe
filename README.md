# Tic-Tac-Toe

J'ai créé ce petit jeu - le fameux Morpion (en français). <br />
Avec la souris, on choisi une case dans un tableau. 
Le but est d'aligner trois fois le même symbole, O ou X. 

## Technos utilisees : 

HTML, CSS, JavaScript (React).
